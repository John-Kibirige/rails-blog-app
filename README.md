# <a name="about-project">Rails Blog App</a>

**Rails Blog App** is a simple blog app that supports all CRUD operations which include, creating a new blog, reading it from the database, updating it and deleting it. It also supports comment addition and likes.

## ğŸ’» Getting Started <a name="getting-started"></a>

> Describe how a new developer could make use of your project.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```sh
 gem install rails
```


### Setup

Clone this repository to your desired folder:

```sh
  cd rails-blog-app
  git@github.com:John-Kibirige/rails-blog-app.git
```

### Install

Install this project with:

```sh
  cd rails-blog-app
  gem install
```

### Usage

To run the project, execute the following command:

```sh
  rails server
```
### Run tests

To run tests, run the following command:

```sh
  bin/rails test test/models/article_test.rb
```
